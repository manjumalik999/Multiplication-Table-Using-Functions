# Multiplication-Table-Using-Functions

📚 Multiplication Table Generator Using Functions (C)

This beginner-friendly C program demonstrates how to use functions to generate the multiplication table of a given number. Instead of writing the logic directly inside main(), the program uses a separate function named myfun() to print the table from 1 to 10, making the code clean, modular, and easy to understand.

✨ Features
🔢 Prints the multiplication table from 1 to 10
🛠️ Demonstrates function creation and function calling
📥 Accepts a number through a function parameter
📖 Beginner-friendly and well-structured code
💻 Great practice for understanding functions in C
🧠 Concepts Used

Functions
Function Parameters
for Loop
Formatted Output (printf)
Basic Program Structure

📌 Sample Output
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
...
2 x 10 = 20

🎯 Learning Outcome

This project helps beginners understand how to organize code using functions, pass arguments, and use loops efficiently to perform repetitive tasks.
